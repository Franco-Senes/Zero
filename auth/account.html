Guia para desarrolladores

Puedes usar Zero como un proveedor de autenticacion OAuth 2.0. A continuacion te indicamos como integrarlo en tus proyectos.

1. Autorizacion

Redirecciona a tus usuarios a la siguiente URL para comenzar el flujo de OAuth:

GET http://localhost:3000/oauth/authorize?client_id=TU_CLIENT_ID&redirect_uri=TU_REDIRECT_URI&state=OPCIONAL

2. Intercambio de Token

Una vez que el usuario consienta, sera redirigido a tu callback con un parametro code. Intercambialo enviando una peticion POST:

POST http://localhost:3000/oauth/token
Content-Type: application/json

{
  "client_id": "TU_CLIENT_ID",
  "client_secret": "TU_CLIENT_SECRET",
  "code": "CODIGO_RECIBIDO",
  "redirect_uri": "TU_REDIRECT_URI"
}

3. Obtener informacion del usuario

Utiliza el access token obtenido para hacer consultas a la API de Zero:

GET http://localhost:3000/api/userinfo
Authorization: Bearer EL_ACCESS_TOKEN_RECIBIDO
